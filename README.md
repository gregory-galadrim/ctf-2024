# ctf-2024

## How to run locally ?

### Requirements

To run this project locally you will need `docker-compose` and `pnpm`

Before all, run

```bash
pnpm install
pnpm steps build
```

### Start the database

```bash
pnpm run dev:db
```

### Start the backend

```bash
pnpm backend dev
```

### Start the frontend

```bash
pnpm ui dev
```
